COMPUTAÇÃO QUÂNTICA

A computação quântica se utiliza das mecânicas quânticas, a junção de todo o trabalho da matemática, fisíca e ciência, com sua abordagem sofisticada para cálculos paralelos, usando as partículas subatômicas para o seu processo, substituindo os transistores dos computadores do nosso dia a dia. Seu objetivo é ter a melhor solução para problemas comparadas a um computador comum, mas também ter uma velocidade nunca já vista para o mesmo.
